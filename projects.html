Project One: Heart Rate Monitor

Course Project
Arduino, C++

This Project was conducted using the Galvanic Skin Theory, which states that By applying a low constant voltage, the change in skin conductance (SC) can be measured non-invasively, to create a model of a heart rate monitor. The project was conducted with the use of an ardukit, which included the arduino microcontroller as well as other componenets. Project Two: Autonomous Robot

Course Project
Robot-C (Variation of C Programming Language), VEX Robotics

'Clamps' was a freshman project. We were required to build a robot to complete a task of our choice. My teammate and I decided to work on a 'pick-up' robot. Its original purpose was to pick up toys around a toddler's playroom, but we realized its purpose could further be expanded as a garbage collecting robot. The image shown is a working prototype of the robot. We intended to 3D print a box matching the dimensions of the one above using the Solidworks software, but decided to keep the more conservative option of a cardboard box.
